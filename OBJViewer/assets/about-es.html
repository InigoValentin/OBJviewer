Obj Viewer



Un visor de modelos tridimensionales Wavefront obj para anadroid.

Incluye

  • Rotacion de modelos.
  • Zoom.
  • Representar (o no) vertices, aristas y caras.
  • Uso de materiales mediante archivos mtl.
  • Cambia el color de los vertices, aristas y caras en tiempo real.
  • Cambia la opacidad del modelo en tiempo real.

Instrucciones

Abre el menu deslizando la pantalla de izquierda a derecha, abre el modelo y listo. Desde el mismo menu pueden elegirse opciones para la vista del modelo.
